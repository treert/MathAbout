### 循环论证的思维茅厕
很容易就陷入了。
比如求$\lim_{n \to \infty} {(1+\frac{1}{n})^n}$。
一个解法是：把n从整数域扩展到实数域，使用自然对数函数，转换后再用洛必达法则求导。
一通计算后，也很完美，求得自然常数e，而且这个方法很通用，好记好掌握！！

这个证明方法就用些像推论反过来证明初始命题。
它错误吗？它正确吗？
